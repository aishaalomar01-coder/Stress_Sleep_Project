# Stress & Sleep Analysis

This project analyzes the relationship between sleep duration and stress levels
using the Sleep Health and Lifestyle dataset from Kaggle.

## Files
- `DataProject.py` – Main Python analysis script
- `project.qmd` – Quarto report source file
- `project.html` / `project.pdf` – Rendered report outputs

## Methods
- Data cleaning and validation
- Pearson correlation analysis
- Independent t-tests
- Data visualization (Seaborn, Matplotlib)

## Dataset
Sleep Health and Lifestyle Dataset (Kaggle)
